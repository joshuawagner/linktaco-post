# LinkTaco API Information Request

To complete polished background-saving support (without opening a visible browser tab), please provide the following:

1. **Create bookmark endpoint**
   - Full URL
   - HTTP method
   - Required headers
   - Example request body
   - Example success/failure responses

2. **Authentication method**
   - API token / OAuth / session cookie / other
   - How token is issued and refreshed
   - Required scopes/permissions

3. **Supported fields for create bookmark**
   - `url`
   - `title`
   - `description`
   - `tags`
   - Any optional fields (folder, privacy, source, etc.)

4. **Tag format**
   - Comma-separated string vs string array
   - Max tag length
   - Max number of tags

5. **Rate limits & constraints**
   - Requests/minute
   - Payload size limits

6. **Search endpoint (future feature)**
   - Endpoint URL + method
   - Query parameters
   - Sort/filter support

7. **Environment details**
   - Production base URL
   - Any sandbox/staging URL

## Optional but useful

- Postman collection or OpenAPI spec
- Official docs link
- A curl example for create bookmark and search
